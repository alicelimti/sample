## 2026-06-08 — FROM 2002 TO 2026 히어로 및 전체 레이아웃 디벨롭

### 참고 레퍼런스
- [대구대학교 70주년 사이트](https://70th.daegu.ac.kr/history) — 대형 타이포그래피 히어로 구조 참고

### 작업 내용

#### 1. 풀스크린 히어로 섹션
- `FROM 2002` (좌) / `TO 2026` (우) 대형 타이포그래피 (최대 13rem)
- 배경: 다크 블루 → 로열 블루 그라데이션 + **그래프지 격자 오버레이**
- 핑크 포인트 `FROM`/`TO` 레이블 + 하단 핑크 액센트 라인
- 입장 애니메이션: `fadeInUp` / `fadeIn` 순차 재생

#### 2. 고정 네비게이션
- 투명 → 스크롤 60px 이상 시 다크 블루로 전환 (`useEffect` + scroll 이벤트)
- 우측 전화번호 hover 시 핑크 컬러 변화

#### 3. 스크롤 힌트
- SCROLL 텍스트 + 흘러내리는 라인 애니메이션 (`scrollDrop` keyframe)

#### 4. 카드 섹션 개선
- hover 시 카드 `translateY(-8px)` 부상 효과
- `section-label` (Curriculum) 소형 영문 레이블 추가
- 푸터 신설: 학원명 · 전화 · 개원연도 · 저작권

## 2026-06-08 — Arctic 팔레트 색감 리뉴얼

### 디자인 방향
북유럽 설경 색감 팔레트 적용 — 차갑고 고요한 아이시 블루 계열

### 변경된 색상 시스템

| 역할 | 변경 전 | 변경 후 |
|------|---------|--------|
| 다크 배경 | `#0d1b3e` | `#1e3a6e` 딥 네이비 |
| 메인 블루 | `#2153a0` 로열 블루 | `#3d5a80` 슬레이트 네이비 |
| 포인트 컬러 | `#e84393` 핫 핑크 | `#c97a6a` 웜 코랄/피치 |
| 보조 컬러 | 중성 그레이 계열 | 퍼리윙클 `#8a9fbc` · 파우더 블루 `#b8cce0` |
| 섹션 배경 | 연한 화이트 | 아이시 파우더 블루 그라데이션 |

### 작업 내용
- `index.css` CSS 변수 전체 교체
- 히어로 배경: 2단 → **4단 대기권 그라데이션** (`#152d58 → #1e3a6e → #2e4e7a → #3d5a80`)
- 그리드 오버레이 색상: 흰색 → 파우더 블루 계열
- 카드 헤더: 각 반별 블루 그라데이션 개별 적용
- CTA 버튼 그림자: 핑크 계열 → 코랄 계열 (`rgba(201,122,106,…)`)
