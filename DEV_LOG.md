# 개발일지

## 2026-06-08 — 프로젝트 초기 세팅

### 작업 내용

#### 1. 리포지토리 클론 및 React 프로젝트 초기화
- GitHub 리포지토리 `alicelimti/sample` 클론
- Vite 8 + React 19 기반 프로젝트 생성
- 의존성 설치 완료 (`npm install`)

#### 2. 디자인 시스템 구축 (`src/index.css`)
- CSS 변수 기반 디자인 토큰 정의
  - **메인 컬러:** 다크 블루(`#0d1b3e`), 로열 블루(`#2153a0`)
  - **포인트 컬러:** 핑크(`#e84393`)
- 글로벌 리셋 및 기본 타이포그래피 설정
- 공통 컴포넌트 스타일 정의 (`.btn-primary`, `.btn-secondary`, `.container`, `.section`)
- 반응형 미디어 쿼리 기준점 설정 (768px)

#### 3. 한국어 사이트 기본 설정 (`index.html`)
- `lang="ko"` 설정
- Google Fonts — Noto Sans KR 로드 (400, 500, 700, 900 weight)
- SEO 메타 태그 추가

#### 4. GitHub Pages 배포 설정
- `vite.config.js`에 `base: '/sample/'` 경로 설정
- `gh-pages` 패키지 설치 및 배포 스크립트 추가
- `npm run deploy` 명령으로 자동 배포

### 기술 결정 사항

| 항목 | 선택 | 이유 |
|------|------|------|
| 번들러 | Vite 8 | 빠른 HMR, 간결한 설정 |
| 스타일링 | 순수 CSS + 변수 | 외부 의존성 최소화, 유지보수 용이 |
| 폰트 | Noto Sans KR | 한국어 가독성 최적화 |
| 배포 | GitHub Pages | 리포지토리와 동일 플랫폼, 무료 |
