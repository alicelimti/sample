# 수학 전문학원 안내 사이트

2002년 개원, 초중고 수학 전문학원 공식 안내 사이트입니다.

## 기술 스택

- **프레임워크:** React 19 + Vite 8
- **스타일:** 순수 CSS (CSS 변수 기반 디자인 시스템)
- **배포:** GitHub Pages

## 개발 실행

```bash
npm install
npm run dev
```

## 빌드 & 배포

```bash
npm run build       # 프로덕션 빌드
npm run deploy      # GitHub Pages 배포
```
